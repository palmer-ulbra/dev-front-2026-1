5. Box-Sizing

Define como o tamanho (width/height) é calculado.

box-sizing: content-box (padrão)

Width não inclui padding e border. Tamanho final = width + padding + border

width: 200px
padding: 20px
border: 5px
Tamanho real: 250px

box-sizing: border-box

Width inclui padding e border. Tamanho final = width

width: 200px
padding: 20px
border: 5px
Tamanho real: 200px

6. Exemplo Prático

Duas boxes com mesma width (200px), mas tamanhos finais diferentes:

content-box

Ocupa mais espaço

border-box
