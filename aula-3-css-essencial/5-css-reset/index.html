CSS Reset

O que é CSS Reset?

Navegadores aplicam estilos padrão diferentes para os elementos HTML. Um CSS Reset remove esses estilos padrão para ter controle total.

CSS Reset Básico

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

Este é o reset mais simples. Remove margin e padding de tudo.

CSS Reset Mais Completo

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  line-height: 1.6;
  font-family: sans-serif;
}

ul, ol {
  list-style: none;
}

a {
  text-decoration: none;
  color: inherit;
}

button, input, textarea {
  font-family: inherit;
  font-size: inherit;
  border: none;
  background: none;
}

img {
  max-width: 100%;
  display: block;
}

Resets CSS Famosos

1. Normalize.css

Mantém os estilos úteis do navegador e normaliza as diferenças entre navegadores. Mais moderno que um reset total.

https://necolas.github.io/normalize.css/

2. Reset CSS (Eric Meyer)

Um dos primeiros e mais usados. Remove todos os estilos padrão.

https://meyerweb.com/eric/tools/css/reset/

3. A Modern CSS Reset (Josh Comeau)

Um css reset mais enxuto.

https://www.joshwcomeau.com/css/custom-css-reset/

4. Reset Customizado

Você pode criar seu próprio reset com apenas o que precisa para seu projeto.

Quando usar CSS Reset?

✓ Use quando:

  • Quer controle total sobre os estilos
  • Precisa de consistência entre navegadores
  • Está começando um projeto do zero

✗ Não precisa quando:

  • Está usando um framework CSS (Bootstrap, Tailwind, etc)
  • Os estilos padrão do navegador já servem
  • É um projeto muito simples
